GitHub-Application Repo Automation

This project was made in collaboration with a group of 3. The main purpose of this project is automate the process of turning an existing project into a gitHub repo. The user must input the file path of desired project, and then input what he want to name the Repo. Then click turn project into repo. This project vastly decreases the time it takes to make a git repo and it takes away all of the steps of knowing the commandline commands. Made with Java using the Swing Library for visual components, and also using the github API client. github repo
